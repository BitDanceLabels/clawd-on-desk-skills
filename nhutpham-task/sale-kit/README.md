## Nguyên tắc bán hàng

1. **Bán cảm xúc trước, tính năng sau** — Bumbee là vibe coding companion, không phải tool năng suất khô khan.
2. **Free tier rộng — Pro tier sâu** — bản open-source MIT vẫn đầy đủ; Pro bán add-on (avatar store, gateway API, team dashboard).
3. **Demo trực tiếp > pitch slide** — bật Bumbee lên 30 giây thấy nhảy là khách hiểu liền.
4. **Target dev đang dùng AI** — không bán cho non-dev, không cố giải thích AI agent là gì.
