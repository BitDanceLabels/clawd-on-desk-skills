# 02 — Khách hàng mục tiêu (ICP & Persona)

## Ideal Customer Profile (ICP)

**Dev/Engineer đang dùng ít nhất 1 AI coding CLI hàng ngày, có máy đủ mạnh để chạy Electron, ưa cá nhân hóa môi trường làm việc.**

Nhanh hơn:
- Có cài Claude Code / Cursor / Copilot CLI / Codex CLI
- Dùng AI agent ≥ 1h/ngày
- Có sở thích sticker, mascot, anime, pixel art, kawaii
- Ngân sách tự chi cho dev tool $5-30/tháng (cá nhân) hoặc seat-based (team)

## 5 Persona chính

### Persona 1 — "Mai, indie hacker dùng Claude Code"
- 26 tuổi, freelancer/indie, làm SaaS solo
- Dùng Claude Code 4-6 tiếng/ngày, multi-session liên tục
- Mê anime, có Live2D vTuber yêu thích
- **Pain**: không biết agent nào đang chạy, miss permission request → break flow
- **Buy trigger**: thấy demo Bumbee juggling khi 2 session chạy → thốt lên "cute quá" → cài luôn
- **Gói phù hợp**: Pro Personal $5/tháng + 1-2 avatar mua lẻ

### Persona 2 — "Hùng, Senior Engineer ở startup"
- 32 tuổi, Tech Lead 8 dev, công ty đang chuẩn bị adopt Cursor + Copilot
- Cần visibility vào việc team dùng AI agent thế nào
- **Pain**: không biết team có lạm dụng/lười dùng AI không, permission audit khó
- **Buy trigger**: Team Dashboard cho thấy session/day, average response time, tool-use category
- **Gói phù hợp**: Bumbee Team $8-15/seat/tháng, gói 8 seats

### Persona 3 — "Linh, dev nữ thích trang trí"
- 24 tuổi, mid-level Frontend, dùng Cursor + Copilot
- Decor desktop với widget, pet sim (Wallpaper Engine, vPet)
- **Pain**: pet desktop hiện tại không liên quan workflow, chỉ trang trí thuần
- **Buy trigger**: pet thật sự "biết" mình đang code, có character áo dài/hanfu đa dạng
- **Gói phù hợp**: Free tier + Avatar Store mua skin lẻ $8-15/skin

### Persona 4 — "Anh Tuấn, Founder AI Startup"
- 35 tuổi, đang xây dev tool/agent platform
- Muốn mascot riêng cho sản phẩm để khác biệt
- **Pain**: vẽ mascot từ đầu mất thời gian, integration UI phức tạp
- **Buy trigger**: White-label OEM — dùng engine Bumbee, gắn nhân vật brand riêng
- **Gói phù hợp**: White-label license $5,000-30,000/năm + custom skin $500-2000

### Persona 5 — "Cộng đồng mod/streamer"
- 22-30 tuổi, dev kiêm streamer code
- Chia sẻ desktop trên Twitch/YouTube khi code
- **Pain**: viewer chán nhìn terminal, cần điểm nhấn visual
- **Buy trigger**: Bumbee xuất hiện cute trên stream, tăng engagement
- **Gói phù hợp**: Pro Personal + Avatar Store + có thể được sponsor làm KOL

## Phân khúc thị trường ưu tiên

### Tier 1 — Việt Nam (gần, dễ tiếp cận)
- Cộng đồng dev VN dùng AI: ~50.000-100.000 active user 2026
- Kênh: Facebook group (Vietdev, Vietnam Developers, J2Team), Reddit r/vietnam_dev, TikTok/YouTube tech VN
- Đặc điểm: thích sản phẩm có nhân vật Á Đông (áo dài, hanfu, chibi), giá nhạy cảm — nên có tier $3-5/tháng
- Ưu thế: artist VN vẽ commission rẻ, nội dung Việt hóa nhanh

### Tier 2 — Đông Á (Nhật, Hàn, Đài Loan, TQ)
- Văn hóa anime/Live2D mạnh, chấp nhận pay-per-character cao
- Kênh: Booth.pm, X (Twitter Nhật), Bilibili
- Cần localization Nhật/Trung và partnership artist local

### Tier 3 — Toàn cầu (US, EU)
- Dev community trên ProductHunt, HackerNews, Reddit r/ClaudeAI, r/cursor
- Mở Pro tier USD pricing
- Ưu thế: open-source MIT đã có sẵn → community traction

### Tier 4 — Enterprise (B2B)
- Công ty dev ≥ 20 người đang adopt AI coding
- Chu kỳ bán dài (3-6 tháng), deal lớn ($5k-50k/năm)
- Cần sales team, không phải self-serve

## Tín hiệu mua hàng (Buying Signals)

Khi thấy lead có những dấu hiệu này → priority cao:
- Đăng ảnh setup desktop có vTuber/Live2D
- Comment trên video Claude Code/Cursor demo
- Star repo open-source về dev tool / pet app
- Đăng "ai đang dùng [tên agent] cho [task]" → đang active
- Đang chia sẻ workflow code AI trên blog/X
- Public profile có hashtag #buildinpublic, #indiehackers, #devlife

## Đối tượng KHÔNG nên target

- Non-dev (designer thuần, manager, biz analyst)
- Dev không dùng AI agent
- User máy cấu hình thấp (< 8GB RAM, không chạy nổi Electron)
- Region không có Claude/Cursor/Copilot phổ biến
- Enterprise có policy cấm app desktop bên thứ 3 (banking, gov)
