## 2. Press Release — English

**Headline:**
> Bumbee on Desk launches — first desktop pet that reacts in real-time to 5 AI coding agents

**Date:** _(fill)_
**Location:** Ho Chi Minh City, Vietnam

---

**Ho Chi Minh City, [Date]** — Bumbee Labs today launches **Bumbee on Desk**, a pixel-art desktop pet that reacts in real-time to five major AI coding agents: Claude Code (Anthropic), Cursor Agent, Copilot CLI (GitHub), Codex CLI (OpenAI), and Gemini CLI (Google). This is the first product to bridge desktop-pet experience with developer visibility tooling for AI coding workflows.

Bumbee sits in the corner of the desktop and reflects the live state of every agent session via 12 distinct animation states — thinking when a user sends a prompt, typing when the agent invokes tools, juggling for multiple concurrent sessions, and a "permission bubble" pop-up when the agent requests permission. Unlike decorative pets (Wallpaper Engine, vPet), Bumbee reads agent hook events directly and maps them to the corresponding animation.

"Coding with AI agents in 2026 is more powerful than ever, but also quieter than ever. Bumbee solves the visibility problem peripherally — you don't have to stare at the pet, but you always know what your agent is doing," said _Nhut Pham_, founder of Bumbee Labs.

**Key features:**

- Universal support for 5 AI coding agents in one app
- 12 pixel-art animation states mapped from real hook events
- Floating permission bubble replacing terminal review (Claude Code)
- Mini mode docks the pet at screen edge for focus
- Open-source MIT, privacy-first (local-only hooks on 127.0.0.1)
- Windows 11, macOS (Intel + Apple Silicon), Ubuntu/Linux supported
- Auto-update via GitHub Releases

**Business model:**

The MIT-licensed open-source core is free forever. Bumbee Labs offers paid tiers:
- **Pro Personal** ($4.99/month): Avatar Store credits + cloud sync
- **Pro+ Avatar Pass** ($9.99/month): unlock all avatars
- **Bumbee Team** ($8/seat/month): Team Dashboard + admin console
- **Bumbee Enterprise** (custom, from $20k/year): on-prem + SSO + SLA
- **Bumbee Avatar Store**: character marketplace (chibi, áo dài, hanfu, cyberpunk, and more)

**2026 roadmap:**

Q2 — Avatar Store launch with 12 initial characters commissioned from Vietnamese artists. Q3 — Team Dashboard + Gateway API. Q4 — Live2D support. Q1 2027 — mobile companion.

## 6. Q&A pre-canned (cho nhà báo)

**Q: Tại sao chọn pixel art chứ không Live2D / 3D ngay từ đầu?**
> Pixel art rẻ, nhanh, ai cũng cute. Live2D đắt và phức tạp, bọn mình để tier 2 trên Avatar Store. Bunny default phải reach max audience trước.

**Q: Bumbee có liên quan đến Anthropic không?**
> Không sponsor, không endorse. Project gốc clawd-on-desk dùng Clawd character của Anthropic với acknowledgment. Bumbee đã rebrand, character mới do team commission, không reuse Clawd asset. Anthropic neither sponsor nor reject.

**Q: Vì sao mở source MIT mà còn monetize?**
> Open core là model đã verify (Sentry, Supabase, GitLab). Core MIT để build community + trust + audit. Pro tier monetize qua experience layer (avatar, dashboard, support) — không gate tính năng kỹ thuật.

**Q: Nguy cơ Anthropic / Cursor / OpenAI tự build pet như Bumbee?**
> Có thể. Nhưng vendor sẽ chỉ build cho agent của họ. Bumbee universal 5 agent — khác. Plus, character đa dạng + community marketplace là moat khó copy nhanh.

**Q: Bumbee xử lý multi-monitor thế nào?**
> Pet nhớ vị trí trên monitor cụ thể. Khi monitor disconnect, clamp về monitor gần nhất. Mini mode dock vào mép phải bất kỳ monitor nào.

**Q: Có roadmap mobile không?**
> Q1 2027 mục tiêu iOS/Android — Bumbee mini hiện state agent từ desktop qua push notification. Chưa hard commit.

**Q: Doanh thu / valuation?**
> Hiện chưa raise round, bootstrap. Số liệu doanh thu chia sẻ trong public update sau Q2.
