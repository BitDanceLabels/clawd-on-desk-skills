## Video 3 — Tutorial Setup 3 phút (YouTube)

**Mục tiêu:** SEO YouTube, organic discovery, activation aid.
**Aspect ratio:** 16:9.
**Format:** Talking head + screen share.

### Outline

**[0-15s — Hook]**

> "Trong 3 phút mình sẽ cài Bumbee on Desk, kết nối với Claude Code, và show pet phản ứng real-time với session AI coding của bạn."

**[15-45s — Download & Install]**

- Lên bumbee.dev, click Download
- Chọn OS (demo Windows)
- Run installer NSIS, next-next-next
- Open app

**[45s-1:30 — Pet xuất hiện + Hook auto-register]**

- Pet bunny xuất hiện góc phải dưới
- Show notification "Hooks registered for Claude Code"
- Mở terminal, chạy `claude --version` confirm OK
- Mở `~/.claude/settings.json`, show entry hook đã append

**[1:30-2:30 — Demo pet phản ứng]**

- Prompt Claude Code: "read this file and summarize"
- Pet thinking → typing → happy bounce khi xong
- Tạo subagent: "use Task tool to research X" → pet juggling
- Cố tình tool nguy hiểm: "run rm test.txt" → bubble pop → click Allow

**[2:30-2:45 — Customization]**

- Right-click pet → menu options
- Resize S/M/L
- Language switch EN/中文
- Enter mini mode demo

**[2:45-3:00 — Outro & CTA]**

> "Bumbee free open source. Pro tier có Avatar Store và cloud sync. Link mô tả. Subscribe nếu bạn vibe được pet này. Comment skin nào nên có tiếp."

## Video 4 — Founder Story 5 phút (YouTube)

**Mục tiêu:** Brand story, build trust, recruit community.
**Format:** B-roll + voice-over founder + screen demo.

### Script (rút gọn)

**[0-30s — Hook]**

> "6 tháng trước, mình quit job FE để build một con pet desktop. Hôm nay 2.3k stars. Đây là câu chuyện."

**[30s-1:30 — Vấn đề cá nhân]**

> "Mình nghiện Claude Code. Mỗi ngày 6-8h. Nhưng có vấn đề: terminal im lặng. Mình prompt cái gì lớn — chờ 30 giây không output — mình không biết nó đang nghĩ hay treo. Permission xin liên tục, mình bị mệt, miss request, agent crash mình mới biết.
>
> Mình nghĩ: 'Phải có ai làm pet desktop tích hợp Claude Code chứ.' Search Google không có. clawd-tank chỉ là pet animation thuần, không hook.
>
> OK, build."

**[1:30-3:00 — Quá trình build]**

> "Tháng đầu: Electron crash trên Windows liên tục — drag-drop bug từ layered focus. 2 tuần debug, ra cách 2-window: render + hit. Tháng 2: vẽ 12 SVG state. Tháng 3: hook architecture, agent registry để extensible.
>
> Tháng 4: tách ra hỗ trợ Codex, Copilot, Gemini, Cursor. Mỗi agent là 1 module config.
>
> Tháng 5: launch v0.1 trên ProductHunt. 800 upvote ngày đầu. HackerNews top 5.
>
> Tháng 6: cộng đồng VN bắt đầu cài, vẽ skin Áo Dài cho pet — đó là khoảnh khắc mình quyết định mở Avatar Store."

**[3:00-4:00 — Lessons learned]**

> "3 bài học:
>
> 1. **Pet không phải decor — là peripheral display.** Người dùng quen, peripheral vision tự thông tin về agent state. Đó là superpower.
>
> 2. **Open source first, monetize qua experience.** Mình không gate tính năng kỹ thuật. Pro tier bán avatar và dashboard. Người dùng không bao giờ feel ép.
>
> 3. **Cộng đồng VN sẵn sàng support sản phẩm Made-in-Vietnam.** Skin áo dài, hanfu — nghệ sĩ Việt commission $80-300, IP của tụi mình. Đây là moat."

**[4:00-5:00 — Roadmap & ask]**

> "Roadmap:
> - Q2: Avatar Store launch 12 skin
> - Q3: Team Dashboard, Gateway API
> - Q4: Live2D support
> - Q1 2027: Mobile companion
>
> Ask: cài thử bumbee.dev. Nếu thích, share. Nếu vibe, contribute trên GitHub. Nếu là artist muốn upload skin Avatar Store, DM Discord.
>
> Cảm ơn đã xem 🐰"

## Production checklist

### Pre-production
- [ ] Confirm asset (SVG, GIF) đã có
- [ ] Record voice-over (founder hoặc actor VN/EN)
- [ ] Music license (Epidemic Sound / Artlist subscription)
- [ ] Screen record demo OK
- [ ] Setup OBS / DaVinci Resolve

### Production
- [ ] Record talking head (cho video 3, 4)
- [ ] Screen capture demo (1080p hoặc 4K downscale)
- [ ] Record vertical 9:16 cho TikTok separate
- [ ] B-roll: dev workspace, terminal close-up, keyboard

### Post-production
- [ ] Edit cut chính
- [ ] Color grade
- [ ] Subtitle EN + VN (whisper auto + manual fix)
- [ ] Sound mix (-14 LUFS for YouTube)
- [ ] Export 1080p H264 (YouTube), 1080x1920 H264 (TikTok)
- [ ] Thumbnail YouTube custom (face + pet + text 5 chữ)

### Distribution
- [ ] Upload YouTube với SEO title/description/tag
- [ ] Cross-post TikTok + Reel + Shorts (1 video, 3 platform)
- [ ] Pin comment với link
- [ ] Reply comment trong 24h đầu
- [ ] Repost lên X/LinkedIn/FB với 1-2 câu intro
