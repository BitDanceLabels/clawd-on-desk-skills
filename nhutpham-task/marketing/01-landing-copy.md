## Section 2 — How it works

**Heading:**
> Pet biết bạn đang làm gì với AI.

3 cột:

### Cột 1: Real-time hook
**Icon:** ⚡
**Title:** Hook real-time, không phải đoán
**Body:** Bumbee bám vào event hook của agent (Claude Code, Cursor, Copilot, Gemini) hoặc poll JSONL log (Codex). Thinking, typing, juggling, error — tất cả map 1-1 từ event thật.

### Cột 2: Permission bubble
**Icon:** 🔔
**Title:** Permission không kẹt trong terminal
**Body:** Khi Claude Code xin quyền, Bumbee bốc bubble nổi góc phải dưới. Allow / Deny / "Always allow Read" — 1 click. Hết câu chuyện switch tab tìm terminal.

### Cột 3: 5 agent universal
**Icon:** 🌐
**Title:** 1 pet, 5 agent, không xung đột
**Body:** Đang dùng nhiều agent cùng lúc? Bumbee tổng hợp state ưu tiên cao nhất. 3 session typing → "building". 2 subagent → "conducting". 1 lỗi → "error" 5 giây ưu tiên hiện trước.
